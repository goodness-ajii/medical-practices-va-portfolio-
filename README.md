## How to Convert These to Final Deliverables

All documents are written in Markdown so you can paste them into any tool. Here's the recommended output format for each:

| Document | Recommended Output | How |
|---|---|---|
| 00 — Knowledge Base | PDF (Google Doc → Download as PDF) | For deep-read clients & interviews |
| 01 — Client Profile | PDF | Internal reference; share if asked |
| 02 — Call Handling SOP | PDF | The "system" centerpiece |
| 03 — Call Scripts | PDF | Often the most-requested document |
| 04 — Patient FAQ | PDF + Notion page (optional) | Easy reference for live calls |
| 05 — Call Log | **Live Google Sheet (view-only link)** | Show clients real working format |
| 06 — Scheduling Workflow | PDF + flowchart in Whimsical/Lucidchart | Embed flowchart image in slides |
| 07 — HIPAA Checklist | PDF | Credibility piece |
| 08 — Voice Demo | **3 audio files** on Google Drive or Vocaroo | Most differentiating deliverable |
| 09 — Weekly Report | 1-page PDF designed in Canva | Show ongoing strategic value |
| 10 — Case Study | **12-slide PDF deck** in Canva or Google Slides | Master document for proposals |

### Quick conversion workflow

1. **Markdown → Google Docs:** Paste content; markdown tables and bullets convert cleanly
2. **Apply styles:** Heading 1 / 2 / 3 to section titles; monospace font for code blocks
3. **Add a cover page** in Canva (or in Google Docs)
4. **File → Download → PDF**
5. **Save in your master Drive folder** with consistent filenames (e.g., `Goodness_Ajii_02_CallHandlingSOP.pdf`)

## Adapting This for Real Clients

When hired by an actual medspa, here's the adaptation timeline I commit to:

### Week 1 (Onboarding)
- Sign BAA, complete HIPAA training, finalize workspace audit
- Replace "Glow Aesthetics" placeholders with the real practice name throughout
- Update 01 Client Profile with the real team, services, hours, software
- Update 03 Call Scripts with the real practice greeting and provider names
- Update 04 Patient FAQ with the practice's real pricing and policies
- Set up the Call Log in the practice's existing Google Workspace
- Test phone system and shadow the existing front desk for at least 1 day

### Week 2
- Begin live phone coverage with a senior staff member on standby
- Submit first weekly report

### Week 3+
- Full coverage during agreed hours
- Ongoing weekly reports
- Quarterly SOP review
