## SLIDE 9 — HIPAA COMPLIANCE

### Headline
**HIPAA Is Non-Negotiable**

### Body
A 100-point self-audit covering:

- Workspace security (private, locked room; headset; no PHI on desk)
- Device security (encryption, MFA, auto-lock, VPN)
- Communication security (only BAA-covered tools)
- Voicemail rules (callback only — no PHI)
- Incident response protocol
- Annual training renewal

### Pull-out
> **Voicemails contain only:**
> First name + practice name + callback number.
> Never treatment details, never appointment specifics.

### Visual notes
- Use a checklist visual on the left (✅ ✅ ✅ ✅)
- Voicemail rule callout on the right in a soft red/cream warning box
- Add a 🔒 padlock icon as the slide accent
