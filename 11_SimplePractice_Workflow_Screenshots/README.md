## File Naming Convention

Every screenshot follows a consistent naming pattern that doubles as a working index:

```
SP_[Number]_Medspa_[Description].png
```

Examples:
- `SP_1_Medspa_Appointment Request Widget.png`
- `SP_8_Medspa_Appointment Calendar Day_Week View.png`
- `SP_27_Booking_Link_Preview.png`

This convention means:
- Files appear in numerical order in any folder browser
- Each filename describes what's shown without needing to open the image
- The `SP_` prefix distinguishes SimplePractice screenshots from other portfolio images
